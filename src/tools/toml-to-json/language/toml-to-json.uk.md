# Документація користувача для онлайн-інструменту перетворення TOML у JSON

## 1. Опис інструменту

TOML у JSON — це зручний онлайн-інструмент, який аналізує дані у форматі TOML та конвертує їх у формат JSON. Користувачі можуть легко вставити або ввести дані у форматі TOML у поле вводу з лівої сторони, і відповідний результат у форматі JSON буде автоматично створено з правого боку.

## 2. Детальний опис функцій

### 1. Введення даних у форматі TOML

* У текстовому полі з лівої сторони під назвою "Your TOML" користувачі можуть вставити або вручну ввести дані у форматі TOML. Це текстове поле підтримує введення багаторядкового тексту й надає достатньо місця для введення складніших TOML-даних.

### 2. Виведення результату у форматі JSON

* Як тільки TOML-дані буде введено або вставлено в текстове поле з лівої сторони, інструмент автоматично виконає їх аналіз та перетворення. Перетворені JSON-дані будуть показані в текстовому полі з правого боку під назвою "JSON from your TOML". Користувачі зможуть переглянути, скопіювати або обробити ці дані далі.

## 3. Кроки використання

1. Відкрийте сторінку інструменту ([https://atoolio.com/toml-to-json](https://atoolio.com/toml-to-json)).
2. У текстове поле з лівої сторони під назвою "Your TOML" вставте або введіть TOML-дані, які потрібно конвертувати.
3. Система автоматично виконає перетворення, і результат буде відображено в текстовому полі з правого боку під назвою "JSON from your TOML".

## 4. Основні особливості

* Простота та інтуїтивність: Інтерфейс зрозумілий і легкий у використанні, процес операцій простий, не потребує складних налаштувань чи конфігурацій, що дозволяє користувачам швидко освоїти роботу з інструментом.
* Конвертація у реальному часі: Після введення TOML-даних інструмент автоматично конвертує їх і показує результат без необхідності натискання кнопки конвертації вручну, що підвищує ефективність перетворень.
* Робота онлайн: Не потрібно встановлювати жодного програмного забезпечення. За наявності пристрою з доступом до Інтернету ви можете скористатися цим інструментом у будь-який момент і в будь-якому місці для перетворення TOML у JSON.

## 5. Приклади

### Приклад 1: Просте перетворення TOML у JSON

Припустимо, у нас є такі дані у форматі TOML:
```toml
title = "TOML Example"
owner = "John Doe"
```

Вставивши ці дані у текстове поле з лівої сторони під назвою "Your TOML", інструмент автоматично перетворить їх у такий формат JSON:
```json
{
  "title": "TOML Example",
  "owner": "John Doe"
}
```

### Приклад 2: Перетворення TOML у JSON із вкладеною структурою

Маємо більш складні TOML-дані:
```toml
[database]
server = "192.168.1.1"
ports = [8001, 8002, 8003]
connection_max = 5000
```

Після їх введення в інструмент отримаємо такі JSON-дані:
```json
{
  "database": {
    "server": "192.168.1.1",
    "ports": [8001, 8002, 8003],
    "connection_max": 5000
  }
}
