# オンラインWiFi QRコードジェネレータードキュメント

## 機能概要

このツールは、ユーザーが迅速にWiFiネットワーク用のQRコードを生成できるように設計されており、他の人がQRコードをスキャンして指定されたWiFiネットワークに簡単に接続できるようにすることを目的としています。

## 機能説明

### 暗号化方式の選択

ユーザーはドロップダウンメニューからWiFiネットワークで使用される暗号化方式を選択できます。「WPA/WPA2」オプションが現在提供されています。

### SSID入力とSSIDの非表示

  * **SSID入力**: 対応する入力ボックスに、WiFiネットワークの名前（つまりSSID）を正確に入力する必要があります。これは他の人が正しいネットワークを認識して接続するために必要な重要な情報です。
  * **SSIDの非表示**: このオプションをチェックすると、WiFiネットワークのSSIDが周囲のデバイスの利用可能なWiFiリストに表示されなくなり、ネットワークの秘匿性が高まります。チェックしない場合は、SSIDが通常通り表示されます。

### パスワード入力

パスワード入力ボックスには、WiFiネットワークのパスワードを入力する必要があります。パスワードを正しく入力した場合のみ、生成されたQRコードを使用してスキャン者がWiFiネットワークに正常に接続できます。

### 前景色と背景色の設定

  * **前景色**: 前景色入力ボックスをクリックすることで、QRコードのパターンの色を選択し、入力できます。たとえば、「#000000ff」は黒を表します。
  * **背景色**: 背景色入力ボックスをクリックすることで、QRコードの背景色を選択し、入力できます。たとえば、「#ffffffff」は白を表します。

上記の各機能項目をカスタマイズすることで、ユーザーは自身のニーズに合わせたWiFi QRコードを生成し、誰でも迅速かつ簡単にWiFiネットワークに接続できるようになります。
